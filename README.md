# WaveNote

### Manage your daily task and workflow in a modern way and boost your effiency without any efforts.

### Features

1. User Signup
2. User Login
3. Add a Note with Priority(Low, Medium, High), Tags (Exercise, Work etc.)

   - Choose a color for the note

4. Edit a Note
5. Archive a Note
6. Trash a Note
